## 문제 설명

예를들어 s가 a234이면 False를 리턴하고 1234라면 True를 리턴하면 됩니다.

## 제한 사항
* s는 길이 1 이상, 길이 8 이하인 문자열입니다.

## 입출력 예
| s|	return|
|--|--|
|a234|	false|
|1234	|true|
